# Think-book

Think-book is a lightweight React application for capturing, organizing, and revisiting ideas. It helps you quickly jot down thoughts, organize them into notebooks and tags, and search or filter entries to support focused thinking and learning.

## Key features

- Create, edit, and delete notes
- Organize notes with notebooks and tags
- Full-text search and filter by tag/notebook
- Responsive UI for desktop and mobile
- Local persistence (and optional sync integration)

## Tech stack

- React (functional components + hooks)
- React Router
- State management: Context API or Redux
- Styling: CSS Modules / Styled Components / Tailwind
- Storage: localStorage (or IndexedDB)

## Quick start

1. Clone the repo
2. Install dependencies:
   npm install
3. Run the app:
   npm start
